Movie Project

Why?

This is a project for Re:Coded Front-End Bootcamp.
It's for practising Object-Oriented JavaScript programming and fetching data from api.
It shows movies, their casts, ratings, trailers, related movies and genres.

How?

This project fetches data from The Movie DB API: https://api.themoviedb.org/3.
